# Legacy Whole-Organism AR Scripts

This directory contains older autoregressive scripts that are no longer the
default development path.

These scripts were moved here because they depend on one or more of:

- synthetic trajectories,
- explicit lineage supervision,
- founder-centric perturbation or masking analyses,
- founder-centric demos or visualization workflows,
- trimodal or crossmodal checkpoints,
- older rollout assumptions that do not match the current roadmap.

Use them only for historical reference or targeted regression checks.

Current entry points live in `examples/whole_organism_ar/`.
